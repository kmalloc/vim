                  <div class="highlight"><pre><div class='line' id='LC1'>==============================================================================</div><div class='line' id='LC2'>CONTENTS                          *simple-bookmarks* *simple-bookmarks-contents*</div><div class='line' id='LC3'><br/></div><div class='line' id='LC4'>&nbsp;&nbsp;&nbsp;&nbsp;Installation...........................: |simple-bookmarks-installation|</div><div class='line' id='LC5'>&nbsp;&nbsp;&nbsp;&nbsp;Usage..................................: |simple-bookmarks-usage|</div><div class='line' id='LC6'>&nbsp;&nbsp;&nbsp;&nbsp;Commands...............................: |simple-bookmarks-commands|</div><div class='line' id='LC7'>&nbsp;&nbsp;&nbsp;&nbsp;Settings...............................: |simple-bookmarks-settings|</div><div class='line' id='LC8'>&nbsp;&nbsp;&nbsp;&nbsp;Issues.................................: |simple-bookmarks-issues|</div><div class='line' id='LC9'><br/></div><div class='line' id='LC10'><br/></div><div class='line' id='LC11'>==============================================================================</div><div class='line' id='LC12'>INSTALLATION                                   *simple-bookmarks-installation*</div><div class='line' id='LC13'><br/></div><div class='line' id='LC14'>There are several ways to install the plugin. The recommended one is by using</div><div class='line' id='LC15'>Tim Pope&#39;s pathogen (http://www.vim.org/scripts/script.php?script_id=2332). In</div><div class='line' id='LC16'>that case, you can clone the plugin&#39;s git repository like so:</div><div class='line' id='LC17'>&gt;</div><div class='line' id='LC18'>&nbsp;&nbsp;&nbsp;&nbsp;git clone git://github.com/AndrewRadev/simple_bookmarks.vim.git ~/.vim/bundle/simple_bookmarks</div><div class='line' id='LC19'>&lt;</div><div class='line' id='LC20'>If your vim configuration is under git version control, you could also set up</div><div class='line' id='LC21'>the repository as a submodule, which would allow you to update more easily.</div><div class='line' id='LC22'>The command is (provided you&#39;re in ~/.vim):</div><div class='line' id='LC23'>&gt;</div><div class='line' id='LC24'>&nbsp;&nbsp;&nbsp;&nbsp;git submodule add git://github.com/AndrewRadev/simple_bookmarks.vim.git bundle/simple_bookmarks</div><div class='line' id='LC25'>&lt;</div><div class='line' id='LC26'>Another way is to simply copy all the essential directories inside the ~/.vim</div><div class='line' id='LC27'>directory: plugin, autoload, doc.</div><div class='line' id='LC28'><br/></div><div class='line' id='LC29'>==============================================================================</div><div class='line' id='LC30'>USAGE                                                 *simple-bookmarks-usage*</div><div class='line' id='LC31'><br/></div><div class='line' id='LC32'>The |:Bookmark| command accepts a name to store the bookmark. It will save the</div><div class='line' id='LC33'>current file and the cursor position to ~/.vim_bookmarks (by default).</div><div class='line' id='LC34'>Afterwards, you can use |:GotoBookmark| with a specific name to jump to that.</div><div class='line' id='LC35'>The command is tab-completed with all known names.</div><div class='line' id='LC36'><br/></div><div class='line' id='LC37'>To delete a mark, use |:DelBookmark|. It works exactly the same as</div><div class='line' id='LC38'>|:GotoBookmark| -- it accepts the name of the mark, which is tab-completed.</div><div class='line' id='LC39'><br/></div><div class='line' id='LC40'>If you&#39;d like to view all your bookmarks, you can use |:CopenBookmarks| to</div><div class='line' id='LC41'>load them in the quickfix window.</div><div class='line' id='LC42'><br/></div><div class='line' id='LC43'>Note that bookmarks should be synchronized between vim instances. Anytime you</div><div class='line' id='LC44'>add a mark in one vim instance, it should be available in all others. In</div><div class='line' id='LC45'>practice, this is achieved by simply reading and writing the whole file on</div><div class='line' id='LC46'>each update. I have yet to experiment to find out if there are any performance</div><div class='line' id='LC47'>issues or race conditions. If you find any issues, please open a bug report on</div><div class='line' id='LC48'>github: https://github.com/AndrewRadev/simple_bookmarks.vim/issues</div><div class='line' id='LC49'><br/></div><div class='line' id='LC50'>==============================================================================</div><div class='line' id='LC51'>COMMANDS                                           *simple-bookmarks-commands*</div><div class='line' id='LC52'><br/></div><div class='line' id='LC53'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*:Bookmark*</div><div class='line' id='LC54'>:Bookmark {name}     Saves the current file and cursor position as a bookmark</div><div class='line' id='LC55'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;with the name {name}.</div><div class='line' id='LC56'><br/></div><div class='line' id='LC57'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*:GotoBookmark*</div><div class='line' id='LC58'>:GotoBookmark {name} Opens up the bookmark associated with the name {name}.</div><div class='line' id='LC59'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Restores the cursor position the way it was at the time</div><div class='line' id='LC60'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;the bookmark was created. Tab-completed with all known</div><div class='line' id='LC61'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;bookmark names.</div><div class='line' id='LC62'><br/></div><div class='line' id='LC63'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*:DelBookmark*</div><div class='line' id='LC64'>:DelBookmark {name}  Deletes the bookmark named {name}. Tab-completed with all</div><div class='line' id='LC65'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;known bookmark names.</div><div class='line' id='LC66'><br/></div><div class='line' id='LC67'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*:CopenBookmarks*</div><div class='line' id='LC68'>:CopenBookmarks      Shows all bookmarks in the quickfix window.</div><div class='line' id='LC69'><br/></div><div class='line' id='LC70'>==============================================================================</div><div class='line' id='LC71'>SETTINGS                                           *simple-bookmarks-settings*</div><div class='line' id='LC72'><br/></div><div class='line' id='LC73'>These are the variables that control the behaviour of the plugin.</div><div class='line' id='LC74'><br/></div><div class='line' id='LC75'><br/></div><div class='line' id='LC76'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*g:simple_bookmarks_filename*</div><div class='line' id='LC77'>&gt;</div><div class='line' id='LC78'>&nbsp;&nbsp;&nbsp;&nbsp;let g:simple_bookmarks_filename = &#39;~/.vim_bookmarks&#39;</div><div class='line' id='LC79'>&lt;</div><div class='line' id='LC80'><br/></div><div class='line' id='LC81'>Default value: &#39;~/.vim_bookmarks&#39;</div><div class='line' id='LC82'><br/></div><div class='line' id='LC83'>This variable is the name of the file that stores the bookmarks. The actual</div><div class='line' id='LC84'>storage format is:</div><div class='line' id='LC85'>&gt;</div><div class='line' id='LC86'>&nbsp;&nbsp;&nbsp;&nbsp;bookmark name \t /path/to/file \t bufnum:lnum:col:off \t bookmarked line</div><div class='line' id='LC87'>&lt;</div><div class='line' id='LC88'>In words, the four parts of a bookmark are tab-delimited, and the</div><div class='line' id='LC89'>second-to-last one, describing the cursor position, contains numbers that are</div><div class='line' id='LC90'>colon-delimited.</div><div class='line' id='LC91'><br/></div><div class='line' id='LC92'><br/></div><div class='line' id='LC93'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*g:simple_bookmarks_long_quickfix*</div><div class='line' id='LC94'>&gt;</div><div class='line' id='LC95'>&nbsp;&nbsp;&nbsp;&nbsp;let g:simple_bookmarks_long_quickfix = 0</div><div class='line' id='LC96'>&lt;</div><div class='line' id='LC97'><br/></div><div class='line' id='LC98'>Default value: 0</div><div class='line' id='LC99'><br/></div><div class='line' id='LC100'>The actual contents of the line are shown next to the bookmark&#39;s name. Setting</div><div class='line' id='LC101'>this variable to 1 would make them go underneath the line instead, making it</div><div class='line' id='LC102'>more readable, but possibly a bit more difficult to navigate.</div><div class='line' id='LC103'><br/></div><div class='line' id='LC104'><br/></div><div class='line' id='LC105'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*g:simple_bookmarks_signs*</div><div class='line' id='LC106'>&gt;</div><div class='line' id='LC107'>&nbsp;&nbsp;&nbsp;&nbsp;let g:simple_bookmarks_signs = 0</div><div class='line' id='LC108'>&lt;</div><div class='line' id='LC109'><br/></div><div class='line' id='LC110'>Default value: 0</div><div class='line' id='LC111'><br/></div><div class='line' id='LC112'>Setting this variable to 1 would make the plugin place &quot;-&gt;&quot; signs on</div><div class='line' id='LC113'>bookmarked lines.</div><div class='line' id='LC114'><br/></div><div class='line' id='LC115'><br/></div><div class='line' id='LC116'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*g:simple_bookmarks_highlight*</div><div class='line' id='LC117'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*SimpleBookmark*</div><div class='line' id='LC118'>&gt;</div><div class='line' id='LC119'>&nbsp;&nbsp;&nbsp;&nbsp;let g:simple_bookmarks_highlight = 0</div><div class='line' id='LC120'>&lt;</div><div class='line' id='LC121'><br/></div><div class='line' id='LC122'>Default value: 0</div><div class='line' id='LC123'><br/></div><div class='line' id='LC124'>Setting this variable to 1 would make the plugin highlight bookmarked lines.</div><div class='line' id='LC125'>The syntax group that&#39;s highlighted is |SimpleBookmark| and it&#39;s linked to</div><div class='line' id='LC126'>|Search| by default. If you want to customize it, you can use Vim&#39;s built-in</div><div class='line' id='LC127'>|:highlight| command.</div><div class='line' id='LC128'><br/></div><div class='line' id='LC129'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*g:simple_bookmarks_auto_close*</div><div class='line' id='LC130'>&gt;</div><div class='line' id='LC131'>&nbsp;&nbsp;&nbsp;&nbsp;let g:simple_bookmarks_auto_close = 1</div><div class='line' id='LC132'>&lt;</div><div class='line' id='LC133'><br/></div><div class='line' id='LC134'>Default value: 1</div><div class='line' id='LC135'><br/></div><div class='line' id='LC136'>If this variable is set to 1, the plugin will automatically close the quickfix</div><div class='line' id='LC137'>window whenever a bookmark is chosen from it. Setting it to 0 disables this</div><div class='line' id='LC138'>special behaviour and leaves the quickfix window open.</div><div class='line' id='LC139'><br/></div><div class='line' id='LC140'><br/></div><div class='line' id='LC141'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*g:simple_bookmarks_new_tab*</div><div class='line' id='LC142'>&gt;</div><div class='line' id='LC143'>&nbsp;&nbsp;&nbsp;&nbsp;let g:simple_bookmarks_new_tab = 1</div><div class='line' id='LC144'>&lt;</div><div class='line' id='LC145'><br/></div><div class='line' id='LC146'>Default value: 0</div><div class='line' id='LC147'><br/></div><div class='line' id='LC148'>If this variable is set to 1, the plugin will open bookmarks in a new tab</div><div class='line' id='LC149'>instead of using the standard quickfix window behaviour.</div><div class='line' id='LC150'><br/></div><div class='line' id='LC151'>==============================================================================</div><div class='line' id='LC152'>ISSUES                                               *simple-bookmarks-issues*</div><div class='line' id='LC153'><br/></div><div class='line' id='LC154'>- Deletion doesn&#39;t prompt before removing a bookmark. Maybe it should.</div><div class='line' id='LC155'>- The storage mechanism is dead simple and it&#39;s possible to break it, though</div><div class='line' id='LC156'>&nbsp;&nbsp;it seems unlikely.</div><div class='line' id='LC157'><br/></div><div class='line' id='LC158'>Any other issues and suggestions are very welcome on the github bugtracker:</div><div class='line' id='LC159'>https://github.com/AndrewRadev/simple_bookmarks.vim/issues</div><div class='line' id='LC160'><br/></div><div class='line' id='LC161'><br/></div><div class='line' id='LC162'>vim:tw=78:sw=4:ft=help:norl:</div></pre></div>
